# ATLAS
ATLAS — Build the future, one command at a time.

ATLAS é um assistente pessoal desenvolvido em Java.

O objetivo do projeto é evoluir gradualmente até se tornar um sistema capaz de automatizar tarefas, gerenciar projetos, interpretar comandos em linguagem natural e integrar diferentes serviços em uma única interface.

## Status do Projeto

🚧 Em desenvolvimento

Versão atual: **0.1.0**

## Funcionalidades atuais

- Interface gráfica em JavaFX
- Dashboard futurista (HUD)
- Relógio em tempo real
- Console interno
- Abertura de aplicativos
- AtlasCore separado da interface
- Contador de aplicativos abertos

## Tecnologias

- Java 25
- JavaFX
- Maven
- CSS

## Objetivos futuros

- Reconhecimento de comandos inteligentes
- Gerenciador de comandos (CommandManager)
- Sistema de IA
- Memória
- Integração com APIs
- Banco de dados
- Automação de tarefas
- Aplicativo Android

## Estrutura do Projeto

- AtlasCore → Núcleo da aplicação
- AtlasApplication → Interface gráfica
- Tasks → Execução de comandos
- Resources → CSS e recursos visuais
